*Developer Tools — fast, single-purpose, CI- and agent-friendly.*

## Why

TOKENMETER exists for one job — token and cost counter / budgeter for llm apps, ci-ready — and does it without a SaaS bill or heavyweight setup.
Single-purpose, scriptable, CI-friendly, self-hostable, and callable by AI agents over MCP.

## Install

```bash
pip install cognis-tokenmeter
# or from this repo:
pip install -e ".[dev]"
```

## Quick start

```bash
tokenmeter --version
tokenmeter scan .                      # scan the current project
tokenmeter scan . --format json
tokenmeter scan . --fail-on high       # non-zero exit for CI gates
tokenmeter mcp                         # expose as an MCP server (Cognis.Studio / Claude Desktop / Cursor)
```
