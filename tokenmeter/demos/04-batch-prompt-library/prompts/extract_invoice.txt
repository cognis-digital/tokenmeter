Extract structured fields from the invoice text and return strict JSON with
exactly these keys: invoice_number, issue_date (YYYY-MM-DD), due_date
(YYYY-MM-DD or null), vendor_name, currency (ISO 4217 code), subtotal, tax,
total, and line_items (array of {description, quantity, unit_price, amount}).

Rules:
- Numbers must be plain decimals with no currency symbols or thousands
  separators.
- If a field is absent in the document, use null. Do not guess.
- Return only the JSON object, no prose, no code fences.

Invoice text:
{document}
